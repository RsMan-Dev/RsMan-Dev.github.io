parses a json string to a table

const a = '{"a":1,"b":2,"c":3, "d": [1, 2, 3]}';


print(JSON.parse(a)); // {a: 1, b: 2, c: 3, d: [1, 2, 3]}

Can parse maps or sets by infering the type

const a = '{"a":1,"b":2,"c":{"e": 1, "f": 2}, "d": {"e": true, "f": false}}';


print(JSON.parse<{
  a: number;
  b: number;
  c: Map<string, number>;
  d: Set<string>;
}>(a)); // {a: 1, b: 2, c: {e: 1, f: 2}, d: {e: true, f: false}}
