the get hook is triggered when a property is accessed on the proxy

```ts
const a = new Proxy({}, {
  get: (target, prop) => {
    print("get", prop);
    return prop;
  }
});

a.a; // will log "get a"

a["a"]; // will log "get a"

a.b = 1; // will log nothing as the set trap is not defined
```

<br/>

Get can be bypassed for certain values, by using the second argument of the Proxy constructor, it will be the main table in setmetatable()

```ts
const a = new Proxy({}, {
  get: (target, prop) => {
    print("get", prop);
    return prop;
  }
}, {
  a: 1
});

print(a.a); // will return 1, logs nothing

print(a.b); // will log "get b" and return undefined
```
