# rbxts-jsnatives

A collection of JavaScript-like APIs for Roblox TypeScript (roblox-ts).

## Overview

This package provides implementations of common JavaScript/TypeScript built-in functions and utilities that are not natively available in Roblox Lua. It aims to make porting JavaScript/TypeScript code to Roblox easier and provide familiar APIs for TypeScript developers working in Roblox.

## Modules

The package includes several modules:

- **Array utilities** - Extra utilities for arrays ([Array-utils](./Array-utils/0-Introduction.mdx))
- **Console** - Logging and debugging utilities ([Console](./Console/0-Introduction.mdx))
- **JSON** - JSON parsing and stringification ([JSON](./JSON/0-Introduction.mdx))
- **Number** - Number constants and methods ([Number](./Number.mdx))
- **Object** - Object manipulation utilities ([Object](./Object/0-Introduction.mdx))
- **Proxy** - Lua-compatible implementation of JS Proxies ([Proxy](./Proxy/0-Introduction.mdx))
- **Set utilities** - Extra utilities for Set objects ([Set-utils](./Set-utils/0-Introduction.mdx))
- **String** - String manipulation methods ([String](./String/0-Introduction.mdx))
- **Timers** - setTimeout and setInterval implementations ([Timers](./Timers/0-Introduction.mdx))

## Standalone Functions

The package also includes standalone utility functions:

- [**encodeUriComponent**](./encodeUriComponent.mdx) - Encodes URI components for safe use in URLs

## Usage

Import specific utilities from the library:

```ts
import { setTimeout, clearTimeout } from "@rbxts/jsnatives";

const timerId = setTimeout(() => {
  print("This runs after 1 second");
}, 1000);

// Cancel the timer if needed
clearTimeout(timerId);
```

Or import all utilities:

```ts
import * as JsNatives from "@rbxts/jsnatives";

JsNatives.console.log("Hello, %s!", "world");
```

## Installation

```bash
npm install @rbxts/jsnatives
```

You also can install it directly from github (in case you want to use a unreleased version):

```bash
npm install @rbxts/jsnatives@github:RsMan-Dev/rbxts-jsnatives
```
