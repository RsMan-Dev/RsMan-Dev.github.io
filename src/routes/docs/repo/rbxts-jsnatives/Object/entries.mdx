get all entries of an object

```ts
const a = {a: 1, b: 2, c: 3};

print(Object.entries(a)); // [["a", 1], ["b", 2], ["c", 3]]
```

<br/>

On proxy, getting entries will trigger key trap, as we are getting values of the object
```ts
const a = new Proxy({}, {
  ownKeys: () => {
    return ["a", "b", "c"];
  },
  get: (target, prop) => {
    print("key trap", prop);
    return prop;
  }
});

print(Object.entries(a)); // [["a", "a"], ["b", "b"], ["c", "c"]] (prints "key trap a", "key trap b", "key trap c")
```

Note that if the proxy returns undefined, the entry will contains only the key as lua's tables can't have undefined values

```ts
const a = new Proxy({}, {
  ownKeys: () => {
    return ["a", "b", "c"];
  },
  get: (target, prop) => {
    print("key trap", prop);
    return prop === "b" ? undefined : prop;
  }
});

print(Object.entries(a)); // [["a", "a"], ["b"], ["c", "c"]]
```
